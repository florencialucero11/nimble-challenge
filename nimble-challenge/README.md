# Nimble Gravity Frontend Challenge

## Description

This project is a frontend application built with React that consumes the Nimble Gravity API.

The application allows a candidate to:
- View available job positions
- Submit a GitHub repository URL to apply for a job

## Tech Stack

- React
- JavaScript (ES6+)
- Fetch API
- Inline styling
- Vite

## Features

- Candidate data fetching
- Job listing
- Job application submission
- Loading and error states
- Basic UI improvements

## Installation

1. Clone the repository:
   git clone https://github.com/your-username/nimble-gravity-challenge.git

2. Install dependencies:
   npm install

3. Run the project:
   npm run dev

## Notes

- Candidate email is configured in App.jsx.
- Basic validation was added for empty repository URL.
- UI styled with modern dark theme approach.
